# Models

3D-printable STL files for the physical paddle shifter hardware — the paddles that
mount to the steering wheel and actuate the shift switches.

---

## What belongs here

- STL files for each printed part (left paddle, right paddle, mounting brackets, switch housings)
- Any source CAD files (`.f3d`, `.step`, `.3mf`, etc.) if available
- A bill of materials for the non-printed hardware (switches, fasteners, magnets, etc.)
